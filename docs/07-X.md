# 07 — X (Twitter) Entegrasyonu

> X ile ilgili yapılacakların tamamı. TrueNetwork’teki `XApiClient` devralınır. Sıralı geliştirmede derinleştirilecek. Tam bağlam: `00-PROJE-Ana-Dokuman.md`.

## 1. Kimlik doğrulama
**OAuth 1.0a** (user context, Read+Write). Gerekli 4 değer (şifreli DB’de, panelden girilir):
`ApiKey (consumer), ApiSecret, AccessToken, AccessTokenSecret`.
Developer kurulumu: developer.x.com → app → User authentication: OAuth 1.0a, Read and Write → Keys and tokens.

## 2. Yayın akışı (XApiClient)
- **Medya upload:** v1.1 `media/upload` (görsel) → `media_id`.
- **Tweet:** v2 `POST /2/tweets` (metin + `media_ids`).
- İmza: OAuth 1.0a imzalı istekler.

## 3. Metin & içerik eşlemesi
- **`ShortX` ≤ 280 karakter** + hashtag (havuzdan seçilir).
- Görsel: her gönderide (AI görsel veya SkiaCard).
- **Link stratejisi (erişim için):** ana tweet **linksiz** (X dış linkli gönderilerin erişimini kısar); blog linki **ayrı reply** (thread) olarak eklenir → erişim cezası ana posta yansımaz. `ReplyWithLink` aç/kapa.

## 4. Limitler & plan
- X API artık **ücretli**; ücretsiz katman paylaşımda çok sınırlı → medyalı düzenli post için **Basic plan** gerekebilir.
- Rate-limit farkındalığı (Redis token-bucket); retry + üstel geri çekilme.
- Idempotency: dönen tweet `ExternalId` saklanır; retry’da çift-post önlenir.

## 5. Ton (AccountVoiceProfile — faz)
X için “kısa ve vurucu” ton profili; kategori tonundan bağımsız hesap tonu.

## 6. Reklam (Promotion — faz)
X’te ücretli tanıtım (kendi kanalımızı büyütme) **Promotion** tarafında; para kazandığımız değil harcadığımız taraf (bkz. 00 §14).
